## HXDG

    Usage: hxdg ...
    
    This script helps creating desktop entries and assigning mimes to them. This
    is fundamental for oAuth authentication and other mechanisms.
    
        show            : Show honored environment variables.
    
    ## Get URIs.
    
        list-uris       : List defined URIs in `~/.config/mimeapps.list`.
        set-uri URI APP : Set APP.
        del-uri URI     : Delete URI.
    
    ## Create apps.
    
        list-apps             : List apps defined in `~/.local/share/applications`.
        add-app OPTS PRG ARGS : Create application shortcut.
           -n NAME       -k KEYWORD    -g CATEGORY
           -c COMMENT    -i ICON       -o PARAM=VALUE
           -d      : Not a terminal program.
           -m MIME : Set mimes. This is required for URLs for example http url
                     `x-scheme-handler/http`.
        del-app NAME          : Delete application.
